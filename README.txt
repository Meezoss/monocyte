
	>> IMPORTANT <<

If the .Vercel.App link is not working, this is the only other way this webpage can be accessed.

1. Open the Monocyte folder through Visual Studio Code. (if the option isn't available, open Visual Studio Code first and open folder in that way)
2. Open Terminal
(NOTE: The Terminal uses Bash command line shell)
3. type in: 'cd monocyte'
4. type in: 'vercel dev'
5. Click on: http://localhost:3000

To close/stop the Terminal simply Ctrl+C.
